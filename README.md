# Hotel Reservation System

This is a Java-based Hotel Reservation System developed using Object-Oriented Programming principles.

## Features
- Create and manage hotel reservations
- Manage guests and identities
- Validate domain values using value objects
- Unit testing using JUnit

## Technologies Used
- Java 17
- Maven
- JUnit 5
- IntelliJ IDEA

## How to Run
1. Open the project in IntelliJ IDEA
2. Run the `Main` class
3. Run tests using Maven or IntelliJ test runner

## Project Structure
- `domain` – Core business entities
- `value` – Value objects with validation
- `test` – Unit tests
